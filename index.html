bespoke.js

このスライドもbespokeを使って作られてます。

以下が必要なのでインストールします。
(インストール方法はググってね)

  • Node.js
  • Bower
  • Grunt
  • YEOMAN

bespoke.jsをインストール

$ npm install -g generator-bespoke // bespokeをモジュールをインストール

テンプレートを作成

$ mkdir presentation-of-newbie
$ cd presentation-of-newbie
$ yo bespoke

色々聞かれますが適当にyesと答えておけば良いと思います。

サーバー起動

$ grunt server

Github Pageに公開

$ grunt deploy
